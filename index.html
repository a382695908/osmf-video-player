OSMF Video Player

+ Variables (click to open)

  • videoURL : String

    Provides a URL to the video file. REQUIRED

  • posterURL : String

    Provides a URL to the poster image/swf. If not provided, video controls will show.

  • autoPlay : Boolean

    If true, video will begin playing when loaded. This is overridden if posterURL is provided. Default = false

  • hideControls : String

    Provides when to hide video controls. Available options are ALWAYS, HOVER, or NEVER. Default = NEVER

  • hideDelay : int

    Number of seconds before controls are hidden. Requires hideControls to be set to HOVER. Default = 5

  • videoScale : String

    Scale mode of video file. Available options are LETTERBOX (always fits video inside), NONE (native size of the video file), STRETCH (stretches video to fill), ZOOM (scales video to fill). Default = LETTERBOX

  • buttonColor : uint

    Color of video control buttons. Default = 0xFFFFFF

  • controlBarColor : uint

    Color of video control background. Default = 0x000000

  • timelineColor : uint

    Color of video control timeline and time text. Default = 0xFFFFFF

+ Keyboard Shortcuts (click to open)

  • Spacebar : Play/Pause
  • Left/Right Arrows : Seek
  • Up/Down Arrows : Volume
  • M Key : Mute
  • F Key : Fullscreen
